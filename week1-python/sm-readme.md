# Student Management System

## Project Description
This is a simple Python-based Student Management System that allows users to add, view, search, and delete student records. Data is stored permanently using JSON file handling.

---

## Features
- Add Student (Name, Age, Semester)
- View All Students
- Search Student by Name
- Delete Student by Name
- Automatic Data Saving using JSON
- Auto Load Data on Program Start

---

## Technologies Used
- Python
- File Handling
- JSON (data storage)
- Loops and Conditional Statements
